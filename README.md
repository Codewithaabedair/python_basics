# 👋 Name & Age Python Program

## 📌 Description

This is a simple Python program that asks the user for their name and age, then prints a friendly message.

---

## 🚀 How It Works

1. The program asks for your name
2. The program asks for your age
3. It prints a message using your answers

---

## 💻 Code

```python
name = input("What is your name? ")
# This stores awnser into a variable
age = input("How old are you? ")
# This stores awnser into a variable
print(f"Hello {name}, you are {age} years old")
# This show text
```

---

## ▶️ How to Run

1. Make sure you have Python installed
2. Save the file as `main.py`
3. Open terminal or command prompt
4. Run:

```
python main.py
```
